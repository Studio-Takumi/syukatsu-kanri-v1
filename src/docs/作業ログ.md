### ディレクトリ構造と基本設定ファイル作成

#### ディレクトリ構造
```
src/
  ├── screens/      # 各画面コンポーネント
  ├── components/   # 再利用可能なUIコンポーネント
  ├── navigation/   # React Navigationの設定
  ├── store/        # Redux関連（slices, store設定）
  ├── types/        # TypeScriptの型定義
  ├── utils/        # ユーティリティ関数
  ├── hooks/        # カスタムフック
  └── docs/         # 作業ログ
```

#### 作成したファイル
- `src/store/index.ts` - Redux Store設定（Redux Persist含む）
- `src/types/index.ts` - 型定義（Application, Status, Company等）
- `src/utils/api.ts` - API設定（ベースURL、fetch関数、エンドポイント定義）
- `src/utils/constants.ts` - 定数（ステータスの色、ラベル等）

**Note**: ステータス仕様は変更の可能性あり（選考通過などが追加されるかも）
